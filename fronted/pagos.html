Módulo de demostración: Este sistema simula el funcionamiento de una pasarela de pago. Los pagos no son reales y se utilizan datos de prueba.

Flujo real en producción:

  1. Cliente selecciona productos → Carrito → Checkout
  2. Sistema crea Payment Intent en Stripe
  3. Cliente ingresa datos de tarjeta (tokenizados)
  4. Stripe procesa el pago y devuelve resultado
  5. Sistema actualiza estado del pedido y stock
